Project Overview

Developed a custom educational content management system (Elmoumen Academy) to provide students and teachers with a structured online learning platform. The system allows admins to manage levels, years, fields, materials, and teachers, while students access videos, PDFs, and books through an intuitive interface.

Key Features

  • User Authentication for students, teachers, and admin
  • Admin Panel to manage:
    • Levels, Years, and Fields
    • Materials (videos, PDFs, books)
    • Teachers, testimonials, and announcements
    • WhatsApp integration for communication
  • Public Site: landing page, course catalog, teacher profiles, contact form, about page
  • Content Tracking: clicks and downloads for books, videos, and materials
  • Responsive Dashboard with clear UI/UX for easy navigation

Technical Implementation

  • Backend: Laravel 10 (MVC architecture) with RESTful API endpoints
  • Frontend: Blade templates + Alpine.js for interactivity
  • Styling: Tailwind CSS + Vite for fast builds and clean UI
  • Database: MySQL with relational schema for users, levels, years, fields, and materials
  • Other Tools: Composer for PHP dependencies, npm for frontend build, Axios for async data handling
  • SEO: Optimized metadata, structured content, and fast-loading pages for visibility
  • Deployment: Configured on Hostinger with custom domain integration

Business Impact

  • Provided a scalable digital academy platform for delivering structured educational content
  • Reduced administrative workload with centralized content management
  • Improved student access to learning resources (videos, books, materials) anytime, anywhere
  • Increased credibility of the academy through a professional, SEO-optimized online presence
